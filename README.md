# AI Meeting Follow-Up Generator

This tool converts meeting notes or transcripts into:
- Summary
- Action items
- Risks
- Follow-up email draft

Built for the hackathon using React + OpenAI.

## How to run locally

1. Clone the repo
2. Install dependencies

npm install

3. Add your OpenAI key

Create a `.env` file:

OPENAI_API_KEY=your_key_here

4. Run the app

npm run dev
